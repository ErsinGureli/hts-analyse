# hts-analyse

**HTS (Historical Traffic Search) analysis application** developed in accordance with BTK (Information and Communication Technologies Authority) documentation.

The project aims to analyze call traffic records, perform location-based evaluations, and transform raw HTS data into meaningful analytical outputs.

## 🚀 Features
- Analysis of HTS call records
- Google Maps API integration
- Location and time-based evaluations
- Modular and extensible architecture
- Docker support for easy deployment

## 🛠️ Technologies
- Java
- Maven
- Google Maps API
- Docker
